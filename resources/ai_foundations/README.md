# AI Foundations for I-O Psychologists

**A starting point for attendees who are new to AI, language models, and agent-based research tools.**

This folder collects orientation materials, recommended readings, and practical guides to help you build a working understanding of the AI technologies used in this tutorial. These documents serve as reference material for the tutorial pipeline -- the main notebook links to them where relevant so you can dive deeper into any concept as needed.

You do not need to read everything here before the session -- but if any of the concepts in the main [README](../../README.md) felt unfamiliar, start here.

## Recommended External Resources

### For Complete Beginners

- **Anthropic's Introduction to LLMs** -- [docs.anthropic.com](https://docs.anthropic.com/) — Clear, non-technical explanation of how language models work and what they can do
- **OpenAI's Getting Started Guide** -- [platform.openai.com/docs](https://platform.openai.com/docs/overview) — Practical walkthrough of using the OpenAI API

### For I-O Psychologists Specifically

- Nguyen, D. C., & Welch, C. (2025). Generative artificial intelligence in qualitative data analysis. *Organizational Research Methods*, advance online publication. — The key paper on epistemic risks of using AI in qualitative research
- Hickman, L., Thapa, S., Tay, L., Cao, M., & Srinivasan, P. (2020). Text preprocessing for text mining in organizational research. *Organizational Research Methods, 25*(1), 114--146. — Foundational text processing concepts relevant to NLP pipelines

### For Understanding Agents

- Anthropic's Claude Agent SDK documentation -- [docs.anthropic.com](https://docs.anthropic.com/) — How agent-based architectures work in practice
- LangChain documentation -- [python.langchain.com](https://python.langchain.com/) — A popular framework for building AI agent pipelines (used as a reference, not a dependency of this tutorial)
